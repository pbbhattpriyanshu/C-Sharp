# C# Learning Playground

This repository is a hands-on C# learning workspace for practicing beginner-level programming concepts through small, focused projects.

## Overview

Each folder in this repository represents a separate lesson or exercise set. The projects are designed to build your understanding of C# gradually, from basic syntax to more structured programming concepts.

## Prerequisites

Before running the examples, make sure you have:

- .NET SDK installed
- A terminal or command prompt available
- A basic understanding of how to open and navigate folders in your system

## Project Structure

- `01-Basics` — basics of C# syntax and first programs
- `02-Terms` — important terminology and core programming ideas
- `03-Math` — numeric expressions and calculations
- `04-StringMethods` — manipulating strings with built-in methods
- `05-ConditonalFlow` — if/else logic and branching
- `06-Operators` — arithmetic, comparison, and logical operators
- `07-Loops` — loops and small game-style examples
- `08-Arrays` — working with arrays
- `09-Functions` — methods, parameters, and reusable logic
- `10-ExceptionHandling` — try/catch patterns and error management
- `11-MultidimensionalArray` — working with multidimensional arrays
- `12-Classes` — class fundamentals and structure
- `13-Objects` — object creation and usage
- `14-OOPs1` — object-oriented programming basics part 1
- `15-OOPs2` — object-oriented programming basics part 2
- `16-OOPs3` — object-oriented programming basics part 3
- `17-OOPs4` — object-oriented programming basics part 4

## Learning Goals

This repository is meant to help you practice:

- C# syntax and structure
- Variables, data types, and operators
- Conditional logic and loops
- Arrays and collections basics
- Methods and function design
- Exception handling and debugging habits

## Run a Project

Open a lesson folder and run this command:

```bash
dotnet run
```

Example:

```bash
cd 01-Basics
dotnet run
```

## Notes

This workspace is intended as a personal practice space for experimenting with C# fundamentals. Work through the folders in order to build confidence step by step and reinforce each concept with hands-on examples.
